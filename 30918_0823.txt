[step2] : 고객과 주문 테이블 조인해서 출력해 보기

select 고객.*, 주문.* from 고객, 주문; // 두 테이블이 고객번호 기준으로 어마하게 출력됨
균형을 위해 큰테이블의 행을 기준으로 출력됨 - 조건이 없다면 쓸모없는 테이블 일 수 있다.

[step3-1] : 조인한 테이블에서 조건을 주어서 출력해보기

SQL> select 고객.고객번호, 고객.이름, 주문.주문번호, 주문.주문가격 from 고객, 주문
  2  where 고객.고객번호 = 주문.고객번호;

[step3-2] : 위 조인 복사해서 new라는 테이블 복사생성하여 출력
SQL> create table New as select 고객.고객번호, 고객.이름, 주문.주문번호, 주문.주문가격 from 고객, 주문
  2  where 고객.고객번호 = 주문.고객번호;

문제1> NEW 테이블에서 주문가격이 5000 이상이 자료의 고객번호와 이름을 출력하시오.
